# Pickup Changes

Luminescent Platinum has changed the Pickup table to better reflect what players will find useful through their playthrough at various points of their journey.

We recommend the player uses Pokétch App 5, "Pokémon List," to keep an eye on whenever a Pickup 'mon gains an item after every battle.

:::tip

Pickup still relies on Pokémon level, but new ranges unlock with further level caps passed. 

:::
